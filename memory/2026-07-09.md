## a11y: RightMenu documentation/bug-report icon links missing aria-label (10:00 UTC)

**Issue found:** `superset-frontend/src/features/home/RightMenu.tsx` lines 753-764 and 770-781 (`superset-frontend/src/features/home/`) ‚Äî the top-right navbar's documentation and bug-report links are icon-only `<StyledAnchor>` (`<a>`) elements with only a `title` attribute, no `aria-label`/`aria-labelledby`/visible text. `title` is not a reliable accessible name for assistive technology.

WCAG 2.1 SC 4.1.2 (Name, Role, Value) ‚Äî Level A.
