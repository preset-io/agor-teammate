### Fix
Branch `fix/series-getitem-iloc-helpers` off `origin/master`.

`superset/models/helpers.py:2292`:
- `value = row[column_index]` → `value = row.iloc[column_index]`

When `df.apply(..., axis=1)` iterates rows, each `row` is a `pd.Series` with string index labels (column names). `row[0]` was resolved positionally, triggering the FutureWarning. `.iloc[column_index]` makes positional intent explicit.
