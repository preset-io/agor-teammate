## a11y: Google Sheets catalog "remove sheet" icon missing aria-label (10:00 UTC)

**Issue found:** `TableCatalog.tsx:73` (`superset-frontend/src/features/databases/DatabaseModal/DatabaseConnectionForm/`) — the `Icons.CloseOutlined` icon shown next to each Google Sheet entry (when 2+ sheets are configured on a database connection) had an `onClick` handler removing that catalog entry, but no `aria-label`/`aria-labelledby`. `BaseIcon.tsx` auto-generates a fallback `aria-label` from the icon's filename ("close"), so screen reader users heard a generic, context-free announcement rather than knowing what the button removes.
