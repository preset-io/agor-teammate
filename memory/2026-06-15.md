# 2026-06-15 — Deprecation Warning Fix Session

## ~16:00 UTC — Deprecation warning found (Datadog)

**Message:**
```
The field `groupby` is deprecated, please use `columns` instead.
```

**Service:** superset, component: app  
**Cluster:** wealthsimple-aws-mpc  
**Timestamps:** 2026-06-14T16:07–16:28 UTC (5 events in Datadog window)

**Root cause:** `ChartDataQueryObjectSchema` defines `groupby` as a deprecated field but had no `@post_load` hook to rename it before `QueryObjectFactory.create(**query_obj)` was called. When `groupby` landed in `**kwargs` of `QueryObject.__init__`, `_rename_deprecated_fields` fired `logger.warning` on every chart data request for any chart saved with the legacy `groupby` key. Same path existed for `granularity_sqla`, `timeseries_limit`, and `timeseries_limit_metric`.
